# 简帖主题生产流程

## 用户故事

### 运营

作为运营，我需要规划设计主题的素材分类，创建主题任务单，安排设计师的主题生产工作，并且审核设计师提交的主题，审核的过程可能是一次通过，或者多次修改。

### 设计师

作为设计师，我需要完成运营下发的生产任务，当我完成制作时，主题作品需要提交到对应的任务单
